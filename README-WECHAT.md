# 微信接入说明

这个项目已经支持接收公众号文本消息，并把消息写入待办系统。

## 你要准备的东西

1. 一个微信入口
   - 测试用：公众号测试号
   - 正式用：公众号

2. 一个公网 HTTPS 地址
   - 微信不能回调本地 `127.0.0.1`
   - 你最后要拿到类似：
     - `https://你的域名/api/wechat/callback`

3. 一个你自己设置的 Token
   - 例如：`my-wechat-todo-token-2026`

## 第一步：配置环境变量

编辑项目根目录下的 `.env`

```env
WECHAT_TOKEN=my-wechat-todo-token-2026
```

`WECHAT_TOKEN` 要和微信后台里填的 Token 完全一致。

## 第二步：把项目跑到公网

你有 2 种常见做法：

### 做法 A：最快测试

用隧道工具把本地服务暴露到公网：

- Cloudflare Tunnel
- ngrok
- frp

本地项目先启动：

```powershell
npm run dev -- --hostname 0.0.0.0 --port 3011
```

然后把 `3011` 映射成一个公网 HTTPS 地址。

## 第三步：在微信后台填写服务器地址

你需要填：

- URL：`https://你的公网域名/api/wechat/callback`
- Token：你 `.env` 里的 `WECHAT_TOKEN`
- 消息加解密方式：先选 `明文模式`

保存成功后，微信会调用一次 GET 校验接口。

## 第四步：测试是否成功

成功后，你给公众号发一条文字，例如：

- `明天下午给 shenyu 回合作报价`
- `5.28 健身`
- `买防晒`

系统会：

1. 收到微信消息
2. 自动解析日期和分类
3. 写入待办数据
4. 在微信里回复你一条确认消息

## 当前已经支持的能力

- 文本消息录入待办
- 自动识别部分日期
  - 今天
  - 明天
  - 后天
  - 下周二 / 下周三等
  - `5.28`
  - `5月28日`
- 自动分类到：
  - 工作
  - 副业
  - 买
  - 生活
  - 灵感

## 当前还没做的

- 微信每日主动提醒
- 微信内查看月历 / 周历
- 更强的自然语言识别
- 多用户隔离后台

## 最推荐的接入顺序

1. 先用测试号接通“微信发文字 -> 网页记待办”
2. 再换成正式公众号
3. 最后再做每日提醒
